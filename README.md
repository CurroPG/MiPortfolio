# Portfolio - Curro Portillo Guerrero

¡Bienvenido a mi portfolio personal! Este proyecto es mi carta de presentación online, donde publico y documento mis proyectos, formación y experiencia como estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM).

🔗 **Enlace:** [https://mi-portfolio-tau-two.vercel.app/](https://mi-portfolio-tau-two.vercel.app/)

## 🚀 Características

- **Diseño Moderno:** Estética oscura estilo "Neon Retrofuturista", con efectos *glassmorphism* (cristal esmerilado) y animaciones fluidas hechas a medida.
- **Multilenguaje:** Soporte y alternancia en tiempo real entre Español e Inglés con un botón de selección (ES/EN).
- **Totalmente Responsive:** Diseño adaptado para ser perfecto tanto en dispositivos móviles, tablets como monitores de escritorio.
- **Sin Frameworks UI:** Creado principalmente con Vanilla JS de manera artesanal y utilizando las API estándar web.

## 🛠 Tecnologías Utilizadas

- **HTML5:** Marcado moderno y semántico.
- **CSS3 (Vanilla):** Variables CSS, Media Queries, Flexbox, y animaciones avanzadas (@keyframes).
- **JavaScript (Vanilla JS):** Gestión del DOM, interacciones dinámicas con el scroll, partículas interactivas de fondo dibujadas en `<canvas>` y el sistema de internacionalización.
- **Librerías externas:** Iconos proporcionados por [Lucide Icons](https://lucide.dev/).

## 🔧 Cómo ejecutarlo localmente

1. Clona este repositorio:
   ```bash
   git clone https://github.com/CurroPG/MiPortfolio.git
   ```
2. Adéntrate en el proyecto mediante terminal o sistema de ventanas:
   ```bash
   cd MiPortfolio
   ```
3. Ejecución: Al no contar con dependientes de Node ni empaquetadores como Webpack, basta con hacer doble click sobre el archivo `index.html` en la estructura de tu sistema o utilizar la extensión **Live Server** con VS Code para disponer de recarga rápida y pruebas locales.
